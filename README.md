# craptionb0t

## Requirements
#### Requires tesseract
See [tesseract wiki](https://github.com/tesseract-ocr/tesseract/wiki)

For OSX
```shell
brew install tesseract
```

#### Requires Python3.6+
For OSX
```shell
brew install python3
```

#### Install required Python packages
```shell
pip install -r requirements.txt
```
